# Setup

## Installation

Install `nuxt-qrcode` in your Nuxt application.

::code-group

```bash [pnpm]
pnpm add nuxt-qrcode
```

```bash [npm]
npm install nuxt-qrcode
```

```bash [bun]
bun add nuxt-qrcode
```

```bash [yarn]
yarn add nuxt-qrcode
```

::


## Getting started

You'll need to add `nuxt-qrcode` to your modules.

```ts
export default defineNuxtConfig({
  modules: [
    'nuxt-qrcode',
  ],
})

```
