## `Qrcode`

The `Qrcode` component only requires a `value` prop to work

::component-code
---
ignore:
  - class
items:
  variant:
    - default
    - pixelated
    - rounded
    - dots
props:
  value: 'https://qrcode.s94.dev'
  variant: default
  class: 'size-full'

---
::

But it also accepts all the same props available at `qrcode.options` in your `nuxt.config.ts`.
